Cara Melanjutkan Download File yang Gagal di Google Chrome

Sedang mendownload atau mengunduh file menggunakan Google Chrome, tiba-tiba browser tersebut crash dan mengakibatkan proses unduh terhenti. Masalahnya adalah saat proses unduh dilanjutkan, hal tersebut tidak bisa terjadi karena pada Chrome jika proses unduh gagal (failed) maka tidak dapat diproses lagi. Tentunya hal tersebut sangat mengesalkan bukan?

Jangan cemas karena terdapat solusinya, dan menariknya memerlukan bantuan dari Firefox. Pastikan Firefox sudah terinstal juga di komputer (PC) atau laptop Anda. Lalu bagaimana cara melanjutkan unduh file yang gagal di Google Chrome Browser?

Berikut ini cara melanjutkan unduh file yang gagal di Google Chrome browser.

• Proses unduh yang gagal akan menyisakan file pada folder unduhan. File tersebut memiliki extension CRDOWNLOAD yang merupakan kepanjangan dari Chrome Download. Jika proses unduh gagal, maka file ini adalah file yang tidak sempurna, tetapi masih dapat digunakan. Untuk melihat nama file, buka folder unduhan pada Chrome (Ctrl+J) lalu lihat namanya. Untuk keperluan langkah ini, asumsi namanya adalah file_ku.crdownload.

• Lalu salin link unduh dari halaman unduhan di Chrome. Klik kanan dan pilih opsi "Copy link address."

• Buka browser Firefox, lalu paste link pada adress bar dan tekan Enter. Dapat dilihat proses unduh mulai dilakukan. Saat mulai berjalan, pause prosesnya. Caranya adalah klik kanan file pada jendela unduh lalu pilih opsi Pause.

• Buka folder tempat dimana Firefox menyimpan file unduhan tersebut, kemudian cek namanya. Bisa jadi namanya berbeda dengan yang ada di Chrome. Hiraukan extension yang ada, lalu salin namanya saja. Katakanlah file yang sedang diunduh di Firefox ini memiliki nama fileku.part, maka cukup salin bagian "fileku" saja.

• Menujulah ke lokasi file unduhan yang gagal di langkah 1 (berisi file fileku.crdownload) dan rename berdasarkan nama file yang Anda salin di langkah 4 (fileku.part). Ubah juga extension file dari CRDOWNLOAD menjadi PART, dengan begitu namanya menjadi "fileku.part." Windows akan memberikan peringatan bahwa perubahan nama file tersebut akan membuat file tak dapat digunakan, hiraukan saja.

• Buka folder tempat file yang didownload via Firefox tersimpan, lalu hapus file sebelumnya. Gantikan dengan "fileku.part."

• Pada jendela unduh, klik kanan proses unduh yang sebelumnya telah dihentikan lalu lanjutkan kembali (resume). Alih-alih mengunduh baru, Firefox akan melanjutkan proses unduh dari yang telah terhenti pada Chrome.
